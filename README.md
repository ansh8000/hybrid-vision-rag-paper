# Hybrid Vision-RAG Framework

This repository contains the LaTeX source code for our research paper:

"Hybrid Vision-RAG Framework for Intelligent Plant Disease Detection and Actionable Recommendation"

## 📌 Overview
This work proposes a hybrid system combining:
- CNN for plant disease detection
- Hybrid retrieval (Dense + BM25)
- Reciprocal Rank Fusion (RRF)
- Cross-encoder reranking
- LLM-based recommendation generation
